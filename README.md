# DNSMASQ-DOCKER
Projeto da disciplina de Redes de Computadores DCCMAPI 2021.1

## Problema
- Despadronização de acesso a urls locais;
- Demora por resolução de nomes.

## Objetivo
Usar o dnsmasq como serviço em docker para resolver os problemas acima.
